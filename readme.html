There is an epilepton (python) model available for the paper:

Naze S, Bernard C, Jirsa V (2015) Computational modeling of seizure dynamics using coupled neuronal networks: factors shaping epileptiform activity. PLoS Comput Biol 11:e1004209

which can be downloaded from the git repository with a command like

git clone https://bitbucket.org/sebnaze/epilepton
